# 🎓 SkillCircle

An interactive platform where individuals can **offer, learn, and trade skills** within their local area. Whether it’s guitar lessons, coding help, yoga training, or language exchange — SkillCircle connects learners and providers in one community-driven platform.

---

## 🌐 Live URL
👉 [SkillCircle Live Demo](https://skill-circle-90721.web.app/)

---

## 🎯 Project Purpose
SkillCircle aims to build a **local learning ecosystem** where anyone can share or gain new skills easily.  
It promotes peer-to-peer learning, local collaboration, and personal growth by connecting learners and mentors based on shared interests.

---

## ✨ Key Features

- 🧑‍🏫 **Skill Listings:** Browse available skills like guitar, yoga, or coding lessons.  
- 💬 **Booking System:** Users can book sessions with local skill providers.  
- ⭐ **Top Rated Providers:** Highlight the best-rated mentors and trainers.  
- ⚙️ **Password Validation:** Strong password rules with form-level error handling.  
- 🔐 **Firebase Authentication:** Register, log in, and reset password securely.  
- 🔄 **React Router Navigation:** Seamless page transitions and protected routes.  
- 🧠 **Interactive UI:** Built with TailwindCSS and Animate.css for smooth visuals.  
- 📣 **Toast Notifications:** Instant feedback on booking or registration actions.  
- 🌀 **Image Slider (Swiper):** Engaging banner showcasing featured instructors.  
- 🏆 **How It Works Section:** Educates users about how to get started easily.
